Get it!

Initialization

$: git config --global user.name "Name Surname"

$: git config --global user.email "yourEmail"


$: git init

Cloning and Pulling

$: git clone http/ssh_ofYourGitRepo

$: git pull http/ssh_ofYourGitRepo

Git Juice

$: git status

$: git add fineName(or --all)

$: git commit -m "yourComment"

$: git push origin master

Branches

$: git branch

$: git checkout branchName

$: git merge branchName
